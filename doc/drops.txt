NAME
----
drops - drops file sharing agent


SYNOPSIS
--------
----
#include <drops.h>

cc ['flags'] 'files' -lzmq -lczmq -lzyre -ldrops ['libraries']
----

DESCRIPTION
-----------

* linkdrops:drops_agent[3] - start and stop drops agent
